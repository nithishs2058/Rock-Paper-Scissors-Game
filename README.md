🎮 Rock Paper Scissors – Python Game

A simple command-line Rock Paper Scissors game built using Python.
The player competes against the computer, which makes random choices using Python’s built-in random module.

📌 About The Project

This project is a beginner-friendly Python game designed to demonstrate basic programming concepts such as:

User input handling

Conditional statements

Loops

Lists

Random number generation

The game runs in the terminal and allows the player to play multiple rounds until they choose to quit.

🎯 Game Rules

🪨 Rock beats Scissors

✂️ Scissors beats Paper

📄 Paper beats Rock

If both the player and computer choose the same option, the round results in a tie.

🚀 Features

Interactive command-line gameplay

Random computer choice generation

Score tracking system

Input validation

Option to quit anytime

Beginner-friendly code structure

🛠️ Technologies Used

Python 3

Built-in random module

▶️ How to Run the Game

Make sure Python is installed on your system.

Download or clone this repository:

git clone https://github.com/your-username/your-repo-name.git

Navigate to the project folder:

cd your-repo-name

Run the game:

python game.py
